My personal archive of algorithms, templates, and problem solutions for competitive programming in C++17.

* **Base Codeforces Rating:** 950
* **Summer 2026 Goal (WSL / Virtual Contests):** Achieve a 1600+ performance level (Expert)
* **Main Goal for 8th Grade:** Winner of the Final Stage of the All-Russian Olympiad in Informatics (All-Rus) for the 9th-grade level (700+/800 points)

## 📁 Repository Structure
* `/templates` — My optimized algorithm templates (Graphs, DP, Computational Geometry, Data Structures).
* `/platforms/codeforces` — Archive of virtual contests and CF Edu courses.
* `/platforms/cses` — Pure algorithmic problems (Sorting, Graphs).
* `/platforms/vosh` — Upsolving of the All-Russian Olympiad in Informatics problems.
* `/platforms/rsosh` — Solutions and upsolving for RSOSH (Russian School Olympiads) contests.

## ⚡ CP Fast I/O Template
All solutions are built using fast I/O `ios_base::sync_with_stdio(false); cin.tie(NULL);` and the `<bits/stdc++.h>` header.
